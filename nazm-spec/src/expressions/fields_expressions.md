## تعبيرات الوصول إلى الحقول

تُستخدم تعبيرات الوصول إلى الحقول لقراءة أو تعديل قيم الخصائص في الهياكل:

```nazm
هيكل شخص {
    الاسم: #متن،
    العمر: ط4،
}

دالة البداية() {


    احجز _شخص: شخص = .شخص {
        الاسم = "أحمد"،
        العمر: 22
    }؛

    اظهر_(_شخص.العمر)؛ // 22

    احجز _شخص2: شخص = .شخص {
        الاسم = "شريف"،
        العمر: 21،
    }؛

    اظهر_(_شخص2.الاسم)؛ // شريف

}
```
