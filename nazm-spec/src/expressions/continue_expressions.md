## تعبيرات 'وصل'

تُستخدم تعبيرات `وصل` في لغة **نظم** لتخطي باقي التكرار الحالي في الحلقة واستكمال التكرار التالي:

```nazm
دالة البداية() {
    احجز متغير م = 0؛

    طالما م < 5 {

        لو م == 3 {
            // إذا كان العدد 3، قم بالتخطي إلى التكرار التالي
            م += 1؛
            وصل؛
        }

        اظهر_(م)؛
        م += 1؛
    }

}
```

نوع تعبيرات `وصل` هو النوع [`!!`]().
